# PrestashopModuleBase
### Usage
``
composer create-project --prefer-dist prestashop/prestashop-module-boilerplate my-module 
``

Or if you're not using packagist.org
``
composer create-project prestashop/prestashop-module-boilerplate my-module --repository="{\"url\": \"git@github.com:vanengers/PrestashopModuleBase.git\", \"type\": \"vcs\"}" --stability=dev --remove-vcs
``
